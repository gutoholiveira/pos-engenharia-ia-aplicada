# 🎓 Pós-Graduação em Engenharia de IA Aplicada
Repositório oficial dos projetos desenvolvidos durante a Pós-Graduação em Engenharia de Inteligência Artificial Aplicada.

Este repositório documenta a evolução técnica, prática e arquitetural na construção de soluções baseadas em IA, LLMs, automação e engenharia de software orientada a dados.
